# extractosm

`extractosm` contains a set of utility functions to extract data from OpenStreetMap. 

## Quick start

### Prerequisites

* **Make**
* **Python** with **uv**

### Installation

```bash
# Install all dependencies
make install
# Install dependencies required to run the example notebook
make install-example
# List helpful targets
make help
```
